# Core Widgets

Bu klasörde uygulama genelinde kullanılan ortak widget'lar bulunur.

## Widget Kategorileri

### Buttons (`buttons/`)
- `AppButton`: Çok amaçlı buton widget'ı (primary, secondary, outline, text, danger tipleri)

### Inputs (`inputs/`)
- `AppTextField`: Özelleştirilebilir metin giriş alanı

### Cards (`cards/`)
- `AppCard`: Standart kart widget'ı

### Loading (`loading/`)
- `AppLoading`: Yükleme göstergesi
- `AppLoadingOverlay`: Tam ekran yükleme overlay'i

### Empty State (`empty/`)
- `AppEmptyState`: Boş durum gösterimi

### Error (`error/`)
- `AppErrorWidget`: Hata gösterimi widget'ı
